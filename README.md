# Care 2030 — Trusted Hospital Visit Assistance

Caring assistance for elderly parents and expectant mothers — from appointment to homecoming.

**Serving:** Kanyakumari · Tirunelveli · Thoothukudi · Madurai

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel

Push to GitHub, then import at https://vercel.com/new

## Tech Stack

- React 18 + Vite
- Custom component system (no external UI library needed)
- Client-side routing
- Mock data layer (ready to connect to a real database)
